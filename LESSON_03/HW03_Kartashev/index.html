  1. Первый уровень
  • Создайте новый проект с HTML и CSS файлами.
  • Вставьте на страницу блок с цитатой (о теге читем тут).
  • Подключите CSS на созданной вами странице 3-мя способами (с помощью тега style, атрибута style и файла стилей style.css).
  • Задайте цитате 3 разных варианта цвета (синий, красный и зеленый соответственно) - каждый разным способом (инлайновые слили, внешний файл и через тег <style>). Проанализируйте, какой из стилей имеет наивысший приоритет.
